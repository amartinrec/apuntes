Mapear unidades

  • Con los siguientes comandos:
    • Comando: mklink
    • Comando: net use
    • Comando: subst
  • Insertando linea en el registro de windows:
    1. Ir a HKEY_LOCAL_MACHINE \ SYSTEM \ CurrentControlSet \ Control \ Session Manager \ DOS Devices
    2. Insertar nuevo valor REG_SZ llamado X: (donde X es la letra de la unidad) con valor el path con formato "\DosDevices\C:\Directorio\Ejemplo"
    Mas información en el artículo: Mapping drive letters to local folders
