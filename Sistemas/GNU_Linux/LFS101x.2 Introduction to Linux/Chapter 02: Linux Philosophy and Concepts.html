Section 5: Linux Distributions

  • Una distribución linux esta compuesta del kernel y un conjunto de herramientas software.
  • Esquema de distribucion linux:
    • Kernel linux
      • El kernel actual, asi como antiguos kernels se encuentran en www.kernel.org
    • Aplicaciones genéricas libres:
      • bash
      • LibreOffice
      • X Windows System
      • ...
    • Aplicaciones especificas de la distribución:
      • Gestor de paquetes
      • Utilidades de configuración
      • ...
    • Aplicaciones propietarias:
      • drivers
      • Adobe Flash Player
      • ...
    • Manuales para:
      • Aplicaciones
      • Comandos
      • Configuracion
      • ...
    • Servicios de soporte, comercial, community, ....
  • Las diferentes distribuciones linux pueden estar basadas en diferentes versiones de kernel. Por ejemplo las distribución RHEL 6 esta basada en la version de kernel 2.6.32, que es antigua pero muy estable. Otras distribuciones se adaptan mas rapidamente las versiones de kernel mas recientes. El kernel no es un propuesta de todo o nada, por ejemplo, RHEL 6 ha incorporado muchas de las mejoras recientes a la versión 2.6.32
