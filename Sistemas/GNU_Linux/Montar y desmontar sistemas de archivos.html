Montar un sistema de archivos

  • Montar un sistema de archivos (file system) es asociarlo a un directorio

Montar temporalmente

  • Se puede hacer temporalmente empleando el comando mount y desmontandolo después con el comando umount

Montar definitivamente

  • Se puede hacer definitivamente modificando el archivo /etc/fstab que es una abreviatura de filesystem table o tabla del sistema de archivos.
  • El archivo /etc/fstab consta de una serie de lineas de 6 campos separados por uno o mas espacios o tabuladores.
  • Las lineas que empiezan con (#) son comentarios y se ignorarán.
  • Significado de los campos:
    • Dispositivo
      • Primera columna
      • Suelen ser nombres de archivo de dispositivo que hacen referencia a discos duros, disquetes, etc

UUID

  • Para saber el UUID de los sistemas de archivos emplear el comando sudo blkid
