Comandos Linux

avconv

Descripción

Obtiene información sobre archivos multimedia o transformalos a otros formatos compatibles.

Sintaxis

$ avconv

Ejemplos

  • $avconv -i <fichero multimedia>

    Obtiene información de un archivo multimedia.

  • $avconv -i prueba.mpeg -vn -ar 44100 -ac 2 -ab 192 -f mp3 audio.mp3

    Extrae el sonido de un fichero mp4

  • $avconv -i video.avi video_transformado.mpeg

    Convierte de un formato de video avi a mpeg

  • $avconv -i video.mpeg video_transformado.avi

    Convierte de un formato de video mpeg a avi

Instalación

$sudo apt-get install avconv

cp

Descripción

Copia ficheros o directorios

Sintaxis

$cp [opciones] origen destino

Opciones

  • -r, -R
    • Copia recursivamente los directorios
  • -p
    • Preserva los atributos de origen en destino. Esto se refiere a la fecha de modificación, fecha de acceso, ID de usuario y grupo, permisos, etc.
  • -u, --update
    • Copia solamente cuando origen es mas nuevo que destino, o bien este último no posee definido tiempo de modificación.

Ejemplos

  • $sudo cp -rp /home/userA/temp /home/userB/backup/temp_de_userA

    • Si la carpeta temp_de_userA no esta creada la crea.
    • Preserva los atributos de directorios y atributos del userA, ya que ponemos la opcion -p, en caso de no ponerlos pondría los atributos de root.
    • Si no ponemos la opcion -r da error "cp: se omite el directorio ..."
