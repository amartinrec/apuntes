Sublime Text

Instalación de Sublime Text en Linux

  1. sudo add-apt-repository ppa:webupd8team/sublime-text-2
  2. sudo apt-get update
  3. sudo apt-get install sublime-text

Desinstalación de Sublime Text en Linux

  1. sudo apt-get remove sublime-text
  2. Luego es necesario borrar la carpeta /home/<usuario>/.config/sublime-text-2 que es donde guarda las configuraciones de la aplicación.

Codificación de los ficheros de texto en Sublime Text

Los ficheros de texto por defecto de sublime text se guardan con la codificación utf-8, los acentos y las ñs no se ven en el navegador, aparece una A mayuscula + un simbolo.

¿Como saber la codificación de un fichero de texto?

  • $file --mime-enconding <fichero>
  • $file -bi <fichero>

¿Como guardar un fichero con otra codificación desde Sublime?

File > Save with Encoding > ISO-8859-1, de este modo los acentos y las ñ se verán.

¿Como cambiar la configuración se Sublime para que guarde en formato iso-8859-1?

Modificar la linea "default_encoding": "Western (ISO 8859-1)",del fichero Preferences.sublime-settings

Cambiar el tema a Nexus

Cambiar el tema de defecto por el tema nexus

  1. Descargar de EleazarCrusader/nexus-theme a :
    • [Windows]C:\Documents and Settings\<Usuario>\Datos de programa\Sublime Text 2\Packages\Theme - nexus
    • [Linux] /home/<Usuario>/Sublime Text 2/Packages/Theme - nexus
    De alguna de las dos siguientes maneras:
    • git clone https://github.com/EleazarCrusader/nexus-theme/ "Theme - Nexus"
    • Descargando el fichero zip de github y descomprimiendo
  2. Abrir en Sublime el fichero : Preferences - Settings - User
  3. Añadir la linea siguiente "theme": "Nexus.sublime-theme"

Visualizar fichero HTML desde Sublime Text en Firefox

  1. "Tools/Build System/New Build System":
  2. "cmd": ["ruta/firefox", "$file"]
  3. Guardar archivo con Firefox.sublime-build
  4. Salir y volver a entrar
  5. Activar Firefox , Tool > Build Systems > Firefox
  6. Pulsar Ctrl-B para ver la pagina HTML en el navegador

Snippets

Crear un snippet

Tools | New Snippet

Formato de un snippet

<snippet>
<content><![CDATA[Aqui va el contenido]]
<tabtrigger>hola</tabtrigger> Palabra que dispara el snippet al pulsar tab
<scope>text.html</scope> Ámbito para limitar donde se disparará el snippet. El ambito para ficheros html es "text.html"
<description></description> Se usa cuando se muestra el snippet en el menu de Snippets, si no se pone descripción muestra el fichero del snippet.
</snippet>

Donde se guardan

Se pueden guardar en la carpeta de cualquier paquete, sin embargo, para simplificar guardar en Packages/User
