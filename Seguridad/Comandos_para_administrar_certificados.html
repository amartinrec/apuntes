Comandos para administrar certificados

Aplicación: certmgr.msc

Entorno de ventanas para gestionar certificados. Permite ver detalles de sus certificados, modificarlos, eliminarlos o solicitar certificados nuevos. Se abre haciendo un inicio - ejecutar - certmgr.msc en sesion de administrador

Info: Información del comando certmgr.msc en la pagina de microsoft

Aplicación: certmgr.exe

Utilidad en linea de comandos para gestionar certificados.
